## âš¡ Getting Started

### Prerequisites

- Node.js >= 18  
- npm or yarn  
- Backend server running (Go backend recommended)  

### Installation

1. Clone the repository:

```bash
git clone https://github.com/EduBank-AI/edubank-frontend.git
cd edubank-frontend
```

2. Install dependencies:

```bash
npm install
```
or
```bash
yarn
```

3. Start the development server:
```bash
npm run dev
```
or
```bash
yarn dev
```

4. Open your browser at http://localhost:3000
